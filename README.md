# MUSICUI

  - music visualiser in c
  - using raylib

## WORK IN PROGRESS

## USAGE

* to build the binary
``` sh
  ./build.sh
```


* to run binary
``` sh
  ./musicui <PATH_TO_MUSIC>
```

- **Linker might struggle to find the dll**
- **Currently only support .dylib i.e. macos**
- **will be adding linux support with .so soon**
